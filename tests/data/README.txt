医疗文本挖掘测试数据集说明

数据集名称：医疗健康知识测试集
创建时间：2024年
数据来源：人工整理的医疗健康知识

文件说明：
1. medical_qa_test.txt - 医疗问答数据，包含30个常见健康问题及其解答
2. health_topics.txt - 健康保健知识，涵盖疾病预防、健康生活方式等30个主题
3. tcm_knowledge.txt - 中医药知识，包含30个中医理论和实践要点

数据特点：
- 总文本数量：90条
- 文本类型：医疗问答、健康知识、中医理论
- 文本长度：每条50-200字
- 涵盖领域：现代医学、预防保健、传统中医

适用于：
- 文本聚类分析
- 词云生成
- 主题建模
- 语义相似度计算
- 医疗文本挖掘算法测试

使用方法：
1. 将整个test_dataset文件夹打包为zip文件
2. 在文本挖掘页面上传数据集
3. 选择聚类方法和聚类数量
4. 运行分析查看结果

注意事项：
- 数据仅供学习测试使用
- 医疗信息仅供参考，不能替代专业医疗建议
