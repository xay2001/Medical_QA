医疗知识文档 - 常见疾病诊疗指南

一、感冒
感冒是最常见的呼吸道疾病，主要由病毒感染引起。症状包括鼻塞、流涕、咳嗽、发热等。

诊断要点：
1. 急性起病，有鼻塞、流涕等上呼吸道症状
2. 可伴有发热、头痛、全身乏力
3. 病程一般7-10天

治疗原则：
1. 对症治疗为主
2. 多休息，多饮水
3. 可服用解热镇痛药如对乙酰氨基酚
4. 抗病毒药物如奥司他韦

预防措施：
1. 勤洗手，戴口罩
2. 避免人群聚集
3. 增强体质，适当运动
4. 接种流感疫苗

二、高血压
高血压是心血管疾病的重要危险因素，需要长期管理。

诊断标准：
1. 收缩压≥140mmHg或舒张压≥90mmHg
2. 非同日3次测量血压均升高
3. 排除继发性高血压

治疗策略：
1. 生活方式干预：低盐饮食、适量运动、戒烟限酒
2. 药物治疗：ACEI/ARB、钙通道阻滞剂、利尿剂
3. 血压目标：一般<140/90mmHg

并发症：
1. 心脏：心肌梗死、心力衰竭
2. 脑血管：脑出血、脑梗死
3. 肾脏：慢性肾病
4. 眼底：视网膜病变

三、糖尿病
糖尿病是胰岛素分泌或作用缺陷引起的代谢性疾病。

分类：
1. 1型糖尿病：胰岛β细胞破坏，胰岛素绝对缺乏
2. 2型糖尿病：胰岛素抵抗和分泌不足
3. 妊娠期糖尿病
4. 特殊类型糖尿病

诊断标准：
1. 空腹血糖≥7.0mmol/L
2. 随机血糖≥11.1mmol/L（有症状）
3. OGTT 2小时血糖≥11.1mmol/L
4. HbA1c≥6.5%

治疗原则：
1. 教育和心理支持
2. 医学营养治疗
3. 运动治疗
4. 药物治疗
5. 血糖监测

药物选择：
1. 二甲双胍：首选药物
2. 磺脲类：促胰岛素分泌
3. α-糖苷酶抑制剂：延缓糖吸收
4. 胰岛素：1型糖尿病必需，2型糖尿病按需

四、心脏病
心脏病包括多种疾病，如冠心病、心律失常、心肌病等。

冠心病：
1. 病因：冠状动脉粥样硬化
2. 症状：胸痛、胸闷、气短
3. 诊断：心电图、冠脉造影
4. 治疗：药物、介入、手术

心律失常：
1. 分类：快速性、缓慢性
2. 症状：心悸、胸闷、晕厥
3. 诊断：心电图、Holter监测
4. 治疗：药物、电复律、起搏器

心力衰竭：
1. 病因：各种心脏病的终末期
2. 症状：呼吸困难、水肿、乏力
3. 分级：NYHA心功能分级
4. 治疗：ACEI/ARB、β受体阻滞剂、利尿剂

五、消化系统疾病
包括胃炎、消化性溃疡、肝炎等常见疾病。

胃炎：
1. 急性胃炎：药物、感染、应激
2. 慢性胃炎：HP感染为主要病因
3. 症状：上腹痛、恶心、呕吐
4. 治疗：PPI、抗HP治疗

消化性溃疡：
1. 胃溃疡、十二指肠溃疡
2. 主要病因：HP感染、NSAIDs
3. 并发症：出血、穿孔、梗阻
4. 治疗：三联或四联抗HP治疗

肝炎：
1. 病毒性肝炎：甲乙丙丁戊型
2. 药物性肝炎：药物损伤
3. 自身免疫性肝炎
4. 治疗：抗病毒、护肝、对症

六、呼吸系统疾病
包括肺炎、哮喘、慢阻肺等。

肺炎：
1. 社区获得性肺炎
2. 医院获得性肺炎
3. 症状：发热、咳嗽、胸痛
4. 治疗：抗生素、支持治疗

哮喘：
1. 慢性气道炎症性疾病
2. 症状：喘息、胸闷、咳嗽
3. 诊断：肺功能检查
4. 治疗：控制药物+缓解药物

慢阻肺：
1. 不完全可逆性气流受限
2. 主要病因：吸烟
3. 症状：慢性咳嗽、咳痰、气短
4. 治疗：戒烟、支气管扩张剂、吸氧
